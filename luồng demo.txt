LUONG DEMO SAN PHAM

0. Mo Docker Desktop.
1. Chay backend container va PostgreSQL container bang Docker Compose.
2. Kiem tra backend tai http://localhost:3000/api/health.
3. Kiem tra blockchain config tai http://localhost:3000/api/blockchain/status.
4. Mo frontend bang Live Server hoac Chrome.
5. Tren frontend, bam Ket noi MetaMask va kiem tra hien vi dang dung.
6. Tren frontend, bam Kiem tra backend va Kiem tra blockchain status.
7. Admin deploy 3 smart contract bang Remix + MetaMask:
   - CertificateNFT
   - PropertyRegistry
   - PropertyTransactionEscrow
8. Copy NFT_ADDRESS, REGISTRY_ADDRESS, ESCROW_ADDRESS vao bien moi truong backend.
9. Restart backend container de nhan contract address moi.
10. Admin cap role:
   - RegistryRole cho PropertyRegistry trong CertificateNFT.
   - EscrowRole cho Escrow trong CertificateNFT.
   - ESCROW_ROLE cho Escrow trong PropertyRegistry.
11. Tren frontend, admin tao profile A va B.
12. Admin register A va B len smart contract, verified = true.
13. Tren frontend, admin tao property cho A.
14. Admin register property len smart contract.
15. PropertyRegistry goi CertificateNFT de mint NFT certificate cho A.
16. Kiem tra ownerOf(tokenId) = vi A.
17. A muon ban nha cho B.
18. Tren frontend, A/admin tao giao dich ban/chuyen nhuong A -> B.
19. A tao sale tren PropertyTransactionEscrow.
20. A approve NFT cho escrow contract.
21. A deposit NFT vao escrow.
22. Kiem tra ownerOf(tokenId) = ESCROW_ADDRESS.
23. Admin web xac minh ho so giao dich.
24. Admin release NFT cho B.
25. B tro thanh chu so huu moi.
26. Kiem tra ownerOf(tokenId) = vi B.
27. Kiem tra verifyOwnership(propertyId, vi B) = true.
28. Kiem tra database:
    - property da doi owner_wallet_address sang vi B.
    - transfer_contract co status released.
    - property_ownership_history co ban ghi transferred.

GHI CHU DOCKER:

1. Docker dong vai tro chay backend va PostgreSQL, khong thay the Remix, MetaMask hoac blockchain.
2. Khi backend chay trong Docker, DATABASE_URL phai dung:
   postgres://postgres:gacon119@postgres:5432/postgres
3. Khong dung localhost:5433 trong backend container.
4. Neu smart contract deploy tren Sepolia, RPC_URL la URL RPC Sepolia.
5. Neu smart contract deploy tren local Hardhat/Ganache cua may Windows, backend container phai dung:
   RPC_URL=http://host.docker.internal:8545
6. Neu port 5433 bi container demo cu chiem, dung container cu hoac doi port compose.
7. Frontend co the chay ngoai Docker bang Live Server hoac npx serve; frontend goi backend container qua http://localhost:3000.
8. Frontend khong chua ADMIN_PRIVATE_KEY va khong ket noi truc tiep PostgreSQL.
9. Truoc khi demo, luon kiem tra:
   - docker ps
   - docker logs blockchain-backend
   - http://localhost:3000/api/health
   - http://localhost:3000/api/blockchain/status
   - frontend mo duoc va ket noi MetaMask


BUOC IPFS THEM VAO DEMO:

1. Trong backend/.env dien PINATA_JWT va IPFS_GATEWAY.
2. Restart backend container.
3. Tao profile va property SQL truoc.
4. Vao trang Bat dong san, chon ID property SQL.
5. Upload anh tai san len IPFS.
6. Bam Tao metadata NFT de tao ipfs://CID_METADATA.
7. Kiem tra certificate_uri cua property da doi thanh ipfs://CID_METADATA.
8. Khi mint NFT, dung certificate_uri nay lam tokenURI.
9. Sau mint, goi ownerOf/tokenURI de kiem tra NFT dang tro den metadata IPFS.
